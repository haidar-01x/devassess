 # DevAssess — AI Developer Knowledge Tester

**AI-Powered Developer Skill Assessment Tool**

A clean, no-signup web app that lets you test your programming knowledge in popular languages. Get randomized questions, answer them, and receive detailed AI-generated feedback on your skill level — including score, strengths, weaknesses, proficiency estimate (Beginner to Expert), and personalized improvement recommendations.

Live Demo: https://code-skill-scout.vercel.app/

## Overview

DevAssess is a focused developer assessment platform. Pick a language, select how many questions you want (5–30), take the quiz, and let AI analyze your answers for an honest skill profile.

No accounts, no login — just start testing.

## Features

- **Zero authentication** — Optional name input, then straight to the quiz
- **Supported Languages**:
  - Python 🐍
  - JavaScript ⚡
  - TypeScript 🔷
  - Java ☕
  - C++ ⚙️
- **Flexible Quiz Length**: 5, 10, 15, 20, 25, or 30 questions
- **Randomized & Unique Questions**: Fresh set every session for better testing and replay value
- **Smart AI Evaluation**:
  - Total score
  - Strengths and weaknesses breakdown
  - Skill level: Beginner / Intermediate / Advanced / Expert
  - Explanations for incorrect answers
  - Targeted topic recommendations to improve
- **Responsive Design**: Works smoothly on mobile and desktop
- **Fast & Global**: Hosted on Vercel for instant loading anywhere

## Tech Stack

- **Frontend Framework**: Next.js (React-based)
- **Deployment**: Vercel
- **Styling**: Tailwind CSS
- **AI Integration**: Lovable (lovable.dev) — powers question generation, evaluation, and feedback
- **Language**: TypeScript (where applicable)

 
## Usage

Go to https://code-skill-scout.vercel.app/
Select your programming language
Choose number of questions
Click Begin Assessment
Answer the questions
Review your AI-generated skill report

Pro tip: While AI-assisted development is fast and powerful, strong fundamentals in core programming concepts (algorithms, data structures, best practices) remain essential for long-term success.
